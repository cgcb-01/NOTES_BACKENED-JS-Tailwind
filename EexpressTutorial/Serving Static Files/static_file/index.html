Serving Static Files

What are Static Files?

Static files are files that do not change on the server side and are sent to the client as-is. Common static files include:
i. HTML files
ii. CSS stylesheets
iii. JavaScript files
iv. Images (JPEG, PNG, SVG)
v. Fonts and videos

Why Serve Static Files?

i. They are essential for building the frontend of a web application.
ii. Improves loading speed by delivering prebuilt content.
iii. Can be cached by browsers for better performance.
